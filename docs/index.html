Installation Guide

🌟 Chrome Web Store (Recommended)

  1. Visit the Chrome Web Store
  2. Search for "X-Proxy" or use the direct link
  3. Click "Add to Chrome"
  4. Confirm the installation when prompted
  5. The X-Proxy extension will appear in your toolbar
🛒 Install from Chrome Web Store

Manual Installation (Developer Mode)

  1. Download or clone the repository from GitHub
  2. Open Chrome and navigate to chrome://extensions/
  3. Enable "Developer mode" in the top right corner
  4. Click "Load unpacked" and select the project folder
  5. The X-Proxy extension will appear in your toolbar

Build from Source

  1. Clone the repository and navigate to the directory
  2. Install dependencies with npm
  3. Build the extension for production
  4. Load the built extension from the dist folder
# Clone the repository git clone https://github.com/helebest/x-proxy.git cd x-proxy # Install dependencies npm install # Build the extension npm run build # The built extension will be in the `dist` folder

Frequently Asked Questions

Is X-Proxy really free?

Yes, X-Proxy is completely free and open-source under the MIT License. There are no premium features, subscriptions, ads, or hidden costs. You can use all features forever without paying anything.

What proxy types are supported?

X-Proxy supports HTTP, HTTPS, and SOCKS5 proxy protocols, covering the most common proxy types used for web browsing. You can easily switch between different proxy types in your profile settings.

How do I switch between proxies?

Simply click the X-Proxy icon in your Chrome toolbar, then click on any saved proxy profile to activate it. Click "System" to return to your default connection without a proxy. Switching takes less than 100ms.

Does X-Proxy collect my data?

No. X-Proxy does not collect, transmit, or store any of your personal data or browsing history. All proxy configurations are stored locally on your device using Chrome's storage API. We prioritize your privacy and security.

Can I use X-Proxy with other Chromium browsers?

Yes! X-Proxy works with all Chromium-based browsers including Microsoft Edge, Brave, Opera, and Vivaldi. Just install it from the Chrome Web Store or load it manually in developer mode.

How many proxy profiles can I save?

You can save unlimited proxy profiles in X-Proxy. There are no restrictions on the number of profiles you can create, making it perfect for users who need to manage multiple proxy configurations.

Where can I get proxy servers?

X-Proxy is a proxy management tool and doesn't provide proxy servers. You can obtain proxy servers from commercial proxy service providers, your organization's IT department, or set up your own proxy server. Always ensure you have permission to use any proxy server.

Is X-Proxy compatible with VPNs?

X-Proxy works at the browser level, while VPNs work at the system level. You can use both simultaneously, but the VPN will typically take precedence for all traffic. For browser-specific proxy routing, use X-Proxy without an active VPN.
